Other Things to Note

  • "But it doesn't work in IE!"
    Use Edge! (Or something else entirely.)
  • Evergreen browsers - FireFox, Chrome, pretty much everybody
  • Edge is still much slower to push patches out, but better than IE was
  • If you can avoid waiting for them, move on without them
  • Everything we looked at is client-side
    Anything "validation" related still needs to be checked on the back-end!
  • HTML5 isn't even done yet and some things are already depricated!
    HGroup, cache.manifest files
